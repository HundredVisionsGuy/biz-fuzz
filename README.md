# Biz Fuzz
This is my demo of a PySide6 Stacked Layout App for my Programming 2 class.

It's to model stacked layouts, writing to and reading from files and various widgets.

## Features:
* Stacked Layout
* Multiple Files to make up a single screen
* QButtonGroups
* Calendar Widgets
* Saving to files
* Reading from files
